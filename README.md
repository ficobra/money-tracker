# Money Tracker

A desktop finance tracking app built with Python and CustomTkinter.

## Features
- Monthly account balance snapshots
- Fixed & variable expense tracking
- Income tracking (fixed and seasonal)
- Investment portfolio monitoring
- Charts and annual overview
- CSV export
- Data backup & restore

## Requirements
- Python 3.13+
- macOS

## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Notes
All data is stored locally on your machine in SQLite.
No internet connection required.
